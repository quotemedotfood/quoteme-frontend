# QuoteMe 🍽️

> Food Distributor Quote Management System

A modern, guest-first web application for managing food distributor quotes with JWT authentication, trial system, and mobile-optimized UI.

## ✨ Key Features

✅ **Guest-first onboarding** - Start without login  
✅ **5 free trial quotes** - No credit card needed  
✅ **JWT authentication** - Secure Railway API integration  
✅ **Mobile optimized** - Swipe gestures, bottom sheets, card views  
✅ **4 main pages** - Customers, Dashboard, New Quote, Quote Builder  
✅ **Beautiful UI** - Brand colors & modern design  

## 🚀 Deploy Now

### Fastest Way (Vercel - 60 seconds)

```bash
# Install Vercel CLI
npm install -g vercel

# Login and deploy
vercel login
vercel
```

That's it! Your app is live. ✨

### Traditional Way (GitHub + Vercel)

1. Export project from Figma Make
2. Push to GitHub
3. Import to Vercel
4. Deploy automatically

See **[QUICKSTART.md](QUICKSTART.md)** for detailed steps.
